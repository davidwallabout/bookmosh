# App Store Content Rights Information

## For App Information Section

### Content Rights Declaration

**Does your app contain, display, or access third-party content?**
Yes

**Third-Party Content Sources:**
- Book metadata (titles, authors, publication dates) from Open Library API
- Book cover images from Open Library Covers API
- All book information is publicly available and used under Open Library's terms of service

**User-Generated Content:**
- Users can create book reviews and ratings
- Users can send messages in book discussion groups (Pits)
- Users can share book recommendations with friends
- All user-generated content is moderated and users can report inappropriate content

## Privacy and Data Usage

**User Data:**
- See Privacy Policy for full details: https://bookmosh.vercel.app/privacy

- Book reading data is private by default (users can choose to make profiles public)
